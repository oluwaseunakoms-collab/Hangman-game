# Hangman Game 🎯

A simple command-line Hangman game built with Python. 
Guess the hidden word one letter at a time before you run out of attempts!

## How to Play
1. Run the game in your terminal
2. A hidden word will be displayed as underscores (e.g. _ _ _ _ _)
3. Guess one letter at a time
4. You have 6 attempts before the game is over
5. Guess all the letters correctly to win!

## Requirements
- Python 3.x

## How to Run
`bash
python hangman.py
